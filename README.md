
# Project Title

Real time collaborative tree editor

# Tech 
MERN

# Client-side 
Change location to client
```bash
cd ./client
```
Install all dependencies
```bash
npm i 
```
run client-side
```bash
npm start
```

#Server-side
Change location to client
```bash
cd ./server
```
Install all dependencies
```bash
npm i 
```
Create database and table in mysql for that use existing query given in 
```bash
path : server/db.script.
```

run client-side
```bash
npm run server
```
